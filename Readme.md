In matlab, right click "lib-power-grid/" folder, choose "Add to path -> selected folders";

Running order:
1. "noise_and_response.m", "noise_and_response_b5over3.m", to generate noise files and corresponding responses' data files.
2. To analyze, run "do_chi2.m", "do_chi2_growing.m", "do_pd_storage.m", "do_H.m", "do_H_vs_num_realizations.m". 
3. some scripts rely on data files generated by other scripts. In this case, please run the prerequisite script as instructed at the beginning of the comment

currently-using-key-data contains basic parameters and optional data for program.
